############ GDN PAPER 1.3B CONFIG #################
# Faithful reproduction of NVlabs' GatedDeltaNet 1.3B (the paper's closest named pure-GDN model).
# Architecture (hidden, layers, ffn, heads, head dims) follows the paper / NVlabs config exactly.
# Backbone differs from our Llama-1B: hidden 2400 (not 2048), so this is a *different* model size.
# Vocab/tokenizer kept as our Llama-3.2 setup (vocab 128256) for benchmark comparability -- the
# paper itself used the Llama2 tokenizer (vocab 32000), so embeddings here are larger than the paper's.
# Source: attn_bench/results/docs/gated-delta-net-config.md (NVlabs/GatedDeltaNet, ICLR 2025)

### Embeddings / input-output:
--make-vocab-size-divisible-by 128      # sizes the model's input / output embedding matrix
# vocab 128256 (our Llama-3.2 tokenizer); paper used Llama2 vocab 32000
# tie_word_embeddings: true -- pass --untie-embeddings-and-output-weights to disable

### Global / model-wide  (paper's 1.3B numbers)
--hidden-size 2400                      # paper 1.3B hidden (n_embd)
--num-layers 16                         # paper 1.3B layers
--normalization RMSNorm                 # paper uses RMSNorm throughout
--norm-epsilon 1e-5
--disable-bias-linear                   # no biases (matches paper / NVlabs defaults)
--transformer-impl transformer_engine
--seq-length 8192                       # runtime only, doesnt affect model size (paper trained at 4K)

### MLP
--ffn-hidden-size 5888                  # paper 1.3B intermediate (~2.45x hidden)
--swiglu                                # SiLU + gating

### Token mixer  (paper's 9-head GDN, expand_k=0.75 / expand_v=1.5)
--experimental-attention-variant gated_delta_net
--linear-attention-freq [1]*16
--linear-num-key-heads 9           # paper num_heads = 9
--linear-num-value-heads 9         # no GQA: num_value_heads = num_key_heads
--linear-key-head-dim 200          # key_dim = 9*200 = 1800 = 0.75*hidden
--linear-value-head-dim 400        # value_dim = 9*400 = 3600 = 1.5*hidden = 2*key_dim
--linear-conv-kernel-dim 4
--num-attention-heads 32            # required to build the model but is not used for GDN
--position-embedding-type none      # GDN ignores positions
--max-position-embeddings 8192      # required by validation (must be >= seq-length), no effect
# Q/K L2-norm, biases, gating are hardcoded in Megatron to the paper's values
